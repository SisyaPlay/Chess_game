# Chess game on Java swing
The game is made on Java Swing and use chess engine StockFish.
# How to Install and Play
1. Open Game folder. 
2. Make sure Java JDK 17 is installed on your PC.
3. Run Build.cmd.
4. Run Run.cmd.
